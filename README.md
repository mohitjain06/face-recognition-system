# Face Recognition Attendance System

An AI-based Face Recognition Attendance System built using Python, OpenCV, Tkinter, and MySQL.

## Features
- Face Detection
- Face Recognition
- Student Management System
- Attendance Management
- GUI Interface using Tkinter
- Train Data Module
- Developer Section
- Help Desk

## Technologies Used
- Python
- OpenCV
- Tkinter
- MySQL
- PIL (Pillow)

## Project Structure
- main.py → Main application file
- student.py → Student management
- attendance.py → Attendance system
- train.py → Train face data
- face_recognition.py → Face detection & recognition
- developer.py → Developer section
- help.py → Help desk

## How to Run
1. Install Python
2. Install required libraries
3. Configure MySQL database
4. Run:
```bash
python main.py
